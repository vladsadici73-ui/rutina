## puffy's planning tool

A soft, pastel-colored weekly planner that makes planning each day feel more peaceful, personal, and a little bit more fun. Features a realistic animated cat companion, week-view, monthly calendar, task tracking, and recurring routines.

## features

- You can add, complete, and delete daily tasks.
- Create recurring routines for your chosen weekdays.
- Interact with calendars by browsing through weeks and months.
- Monitor your daily progress visually using the completion ring.
- Store all of your planner data right inside your web browser.
- A responsive user interface made of pastels and animated cats.

## how do i use this thing?

1. Open `index.html` or `rutina-pastel.html` in a current-gen web browser.
2. Select a day on the week strip or the monthly calendar.
3. Type in a new task into the input box and click "Add".
4. Mark a task as complete by clicking on the round checkbox.
5. Create recurring tasks in the "Routines" section for the specific days you want them to appear.
## Notes

The tasks & routines that you create on this app are saved locally within your web browser. Therefore, they remain private to you and the computer you use. However, if you clear your browser's data, all of your saved information will be lost.
